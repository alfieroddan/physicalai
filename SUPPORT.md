# Support

## Before opening an issue

Please check the following first:

- The [README](./README.md) for installation and project overview
- The [design docs](./docs/design/README.md) for architecture and interface details
- The [CONTRIBUTING guide](./CONTRIBUTING.md) for development workflow expectations
- Existing [issues](https://github.com/openvinotoolkit/physicalai/issues) for duplicates or related discussions

## Where to ask for help

- Bug reports: open a bug report issue with a minimal reproduction
- Feature ideas: open a feature request issue with the problem and proposed solution
- Usage questions: open a usage question issue and include environment details and logs

## Security issues

Do not use public issues for security reports. Follow the process in [SECURITY.md](./SECURITY.md).
